# Aron.MaplestoryArtale - 楓之谷 Artale 遊戲輔助系統

這是一個專為 MapleStory Artale 設計的遊戲輔助工具，透過 OBS 擷取遊戲畫面，並利用 AI 模型進行影像辨識，實現自動喝水（HP/MP 恢復）與狀態監控功能。

![主介面預覽](Screenshots/01.png)

## ✨ 功能介紹

### 1. 角色狀態監控 (Character Status)
*   **即時數值顯示**：透過圖形化介面即時顯示角色的 HP、MP 與經驗值 (EXP) 百分比。
*   **遊戲聚焦狀態**：顯示目前遊戲視窗是否處於前景（聚焦狀態）。

### 2. 自動化設定 (Automation Settings)
*   **自動 HP 恢復**：
    *   可自訂觸發喝水的血量百分比（例如：低於 70% 時喝水）。
    *   可設定對應的鍵盤按鍵（例如：End 鍵）。
    *   **靈敏度調整**：提供 1-3 級靈敏度設定，避免因畫面波動造成誤判或重複喝水。
*   **自動 MP 恢復**：
    *   可自訂觸發喝水的魔力百分比。
    *   可設定對應的鍵盤按鍵（例如：PageDown 鍵）。
    *   同樣支援靈敏度調整。

### 3. 遊戲畫面預覽 (Game Screen Preview)
*   顯示從 OBS 擷取到的即時遊戲畫面。
*   **偵測標記**：在畫面上方標示出 AI 偵測到的血條、魔力條與經驗條位置（紅色/藍色/綠色框）。
*   提供截圖、查看進程與診斷功能。

### 4. 系統與恢復日誌 (Logs)
*   **系統日誌**：記錄程式運行狀態、OBS 連線狀態、偵測到的遊戲視窗尺寸等資訊。
*   **自動恢復日誌**：記錄每一次自動喝水的時間點與觸發原因，方便回溯與調整設定。

### 5. 偵測結果詳細資訊 (Detection Results)
*   顯示 AI 模型對各個目標（HP Bar, MP Bar, EXP Bar, Level）的辨識信心度 (Confidence) 與座標位置。

## 🚀 使用說明

### 前置作業
本程式已內建 **OBS Portable (免安裝版)**，您 **無需** 額外安裝 OBS Studio 或進行繁瑣的 WebSocket 設定。

### 啟動程式
1.  前往編譯後的目錄：`bin\Release\Aron.MaplestoryArtale`。
2.  執行 `Aron.MaplestoryArtale.exe`。
3.  程式啟動時會 **自動啟動內建的 OBS**，並自動開啟網頁介面（預設為 `http://localhost:5000`，請依實際 Console 顯示為主）。

### 關閉程式
*   直接關閉開啟的 Console (黑窗) 視窗，或在 Console 中按 `Ctrl + C` 即可結束程式。
*   程式關閉時，會 **自動關閉** 剛才啟動的 OBS 程序，無需手動操作。

---

## 🔗 程式與 OBS 的關聯

本程式並非直接讀取遊戲記憶體，而是採用 **非侵入式** 的影像辨識技術。為了簡化使用者操作，程式目錄下已包含 OBS 免安裝檔：

1.  **自動管理**：程式啟動時會自動執行內建的 OBS，結束時也會自動關閉它。
2.  **影像來源**：程式透過 WebSocket 協定與 OBS 通訊，請求 OBS 提供當前的遊戲畫面截圖。
2.  **AI 分析**：程式接收到截圖後，使用內建的模型分析畫面，找出 HP/MP 條的位置與長度。
3.  **模擬按鍵**：當分析結果低於設定的閾值時，程式會模擬鍵盤輸入（發送按鍵訊號），執行喝水動作。

> **注意**：為了確保運作正常，請勿將 OBS 最小化（可以縮小到工作列，但不能完全關閉顯示），並確保遊戲視窗未被其他視窗完全遮擋。
